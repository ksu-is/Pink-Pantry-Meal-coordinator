# Pink Pantry Meal Coordinator – Project Roadmap

## Project Goal
The goal of this project is to create a Python application that helps users enter ingredients they already have at home and receive simple meal suggestions based on those ingredients. The program will also provide basic recipe ideas to reduce food waste and make meal planning easier.

## Sprint 1 Completed
- Created GitHub repository inside KSU-IS organization
- Added README.md with project description
- Researched similar Python recipe repositories
- Reviewed beginner-friendly Python examples

## Sprint 2 Goals
- Create the main Python file
- Build ingredient input system
- Store ingredients using Python lists
- Create simple meal suggestion logic
- Display meal recommendations

## Future Improvements
- Add more recipe options
- Add grocery list suggestions
- Add breakfast, lunch, and dinner categories
- Improve user experience and formatting
